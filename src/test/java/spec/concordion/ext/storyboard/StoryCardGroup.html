Collapsible Group Card

Sometimes your test might perform a large number of steps and the storyboard becomes unweildy. Adding cards to a collapsible group will allow you to click on the first card to show or hide the rest of the cards in the group.

Behaviour:

  • A collapsible group may be started by calling the startGroup() or addCard(...) methods
  • A collapsible group may be stopped by calling the stopGroup() or addCard(...) methods
  • If a card with a failure or warning status is added to the group then the start and stop group cards will be given that same status
  • If the group is not stopped it will be left expanded, if it is stopped then it will be rendered collapsed
  • Clicking on the collapsible group card image will show or hide the rest of the cards in the group

Example 1: Start Group

<span concordion:execute="startGroup(#TEXT)">Setup</span>

When this snippet is run with this fixture, this results in a collapsible start card being added to the Storyboard and the group left expanded.

Example 2: Stop Group

<span concordion:execute="stopGroup(#TEXT)">Setup</span>

When this snippet is run with this fixture, this results in a collapsible stop card being added to the Storyboard and the group rendered collapsed.

Example 3: Add Failure Card to Group

<span concordion:execute="addFailureToGroup(#TEXT)">Setup</span>

When this snippet is run with this fixture, this results the start and stop cards getting a failure status.
